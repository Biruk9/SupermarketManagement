# Supermarket Management System

A simple C++ console-based supermarket management system that allows an administrator to manage products and a buyer to make purchases.

## Features
- Administrator functionalities:
  - Add new products
  - Edit existing products
  - Delete products
  - View product list
- Buyer functionalities:
  - Browse products
  - Purchase multiple products
  - View receipt with total and discounted prices

## File
- `market.cpp` - Main C++ source code file containing all the functionality.

## How to Run
1. Clone the repository:
git clone https://github.com/Biruk9/SupermarketManagement.git
2. Navigate to the project folder and compile the code using a C++ compiler:
g++ market.cpp -o supermarket
3. Run the executable on Windows:
.\supermarket.exe

## Technologies Used
- C++
- File handling (for storing product records)
